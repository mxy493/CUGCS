##### 1.16 比较两个整数a和b的大小

```c++
char Compare(int a, int b)
{
	if (a > b) return '>';
	else if (a == b) return '=';
	else return '<';
}
```

关键操作：不含循环，程序步数为常数

时间复杂度：T(n)=O(1)



##### 1.17 统计各区间元素个数

```c++
void Count(int a[],int size)    //传入数组以及数组大小
{
	int c1, c2, c3, c4, c5;     //记录各区间元素个数
	c1 = c2 = c3 = c4 = c5 = 0;
	for (int i = 0; i < size; i++)
	{
		if (a[i] >= 0 && a[i] <= 20) c1++;
		else if (a[i] <= 50) c2++;
		else if (a[i] <= 80) c3++;
		else if (a[i] <= 130) c4++;
		else c5++;
	}
	cout << "[0,20]:" << c1 << endl;    //输出
	cout << "[21,50]:" << c2 << endl;
	cout << "[51,80]:" << c3 << endl;
	cout << "[81,130]:" << c4 << endl;
	cout << "[131,200]:" << c5 << endl;
}

```
